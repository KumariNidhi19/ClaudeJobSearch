# Application Tracker

Updated daily by the Cowork scheduled task. New matches are appended to the top
of the **Inbox** table (between the two HTML comment markers). Move rows down
into **Pipeline** as you act on them. Before adding, Claude reads this whole file
and skips any posting already listed (same URL, or same company + title).

**Status legend:** `new` → `to-apply` → `applied` → `screening` → `interview`
→ `offer` / `rejected` / `skipped`

**Tier legend:** `1` = 2–3 day hybrid · `2` = 5-day week · `3` = allowed
top-tier exception
