# Cowork Setup — ready-to-paste

Two short things to paste into Cowork. Full walkthrough is in `README.md`.

---

## 1. Global instructions (Settings → Cowork → Global instructions → Edit)

> I am a Product Designer (AI & enterprise systems) with ~4 years' experience,
> Pune preferred but open to all locations, job-hunting. When doing job searches,
> always apply the work-schedule rules in my job-preferences file strictly: 2–3
> day hybrid roles are top priority, then standard 5-day weeks; reject 6-day or
> alternate-Saturday roles unless the company is BCG / JPMorgan / PwC / Deloitte /
> Capgemini tier. Never invent job postings or links. Keep my personal contact
> details private and out of any public file.

## 2. The daily scheduled task

Open a new Cowork task and type `/schedule`, then paste the prompt below (same as
`SCHEDULED_TASK` in `prompts.md`). Choose **Daily** at a time you like
(e.g. 8:30 AM). Confirm.

```
Read profile.md, resume.md, and job-preferences.md in my Job Search project (or
connected GitHub repo). Search the web for currently open Product Designer / UI-UX
/ UX / AI Product Designer roles that fit.

Locations: open to ALL locations, Pune preferred — don't exclude any city or
country. Include on-site, hybrid, and remote.

Search all of these each run: LinkedIn Jobs, Naukri, Indeed India, Wellfound,
Instahyre, Cutshort, Hirist, Foundit, Glassdoor, and YC "Work at a Startup" —
PLUS the careers pages of prominent product-design employers biased to my domains
(AI/GenAI, enterprise SaaS, pharma/health, SaaS): e.g. Google, Microsoft, Adobe,
Atlassian, Salesforce, Intuit, Uber, Amazon, Canva, Figma, Notion, Postman,
Freshworks, Zoho, Razorpay, CRED, PhonePe, Groww, Flipkart, Swiggy, Zomato,
Sprinklr, BrowserStack, ThoughtSpot, Innovaccer, Fractal, Zeta, and other strong
companies. Focus on roles posted/updated in the last 14–21 days.

Apply my schedule rules strictly: 2–3 day hybrid highest (Tier 1), then 5-day
weeks (Tier 2); reject 6-day / alternate-Saturday roles unless the employer is
BCG, JPMorgan Chase, PwC, Deloitte, Capgemini, or comparable. If schedule isn't
stated, include but mark "unknown" and flag it.

Read the existing application-tracker.md first and skip anything already listed
(same URL, or same company + title). Target 20 new roles; if 20 real qualifying
roles don't exist today, add as many genuine ones as you find — never invent
postings or links to reach 20.

Append new roles to the Inbox table, best fit first, each with: date found, tier,
title, company as a markdown link to its careers page, location, work mode,
schedule, fit score 0–100, a short flag, the direct job-posting link, and status
"new". ALSO print the new roles as a clickable markdown table directly in your
reply (Tier · Title · Company linked · Location · Mode · Schedule · Fit · Apply
link · Flag) so I can apply from chat. Save the tracker file and add a 3-line
summary under the table: roles added, best match, and any flagged "unknown".
```

That's it. Each morning you'll get ~20 fresh Inbox rows in
`application-tracker.md` plus a short summary notification.
